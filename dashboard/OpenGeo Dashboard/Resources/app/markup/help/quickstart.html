Getting Started

This page contains a summary of how to get your data published on the web using the OpenGeo Suite. For more information, please see the Basic Workflow section of the OpenGeo Suite User Manual.

Step 1: Load your data

Use pgShapeLoader to convert local shapefiles to tables in your PostGIS database. View your data with pgAdmin.

  • If you are looking for data to get started, the Natural Earth Data has free shapefiles available for download.

Step 2: Publish your data

Use the Layer Importer to load your database tables into GeoServer. Each database table will become an individual map layer.

Step 3: Compose your map

Use GeoExplorer to add layers served through your GeoServer to a map, along with content from a wide variety of other hosted map services.

Step 4: Edit your data

Use GeoExplorer to create and edit features graphically, right in the browser, on any layers served through GeoServer.

Step 5: Style your map layers

Use GeoExplorer and its built-in graphical styling tools to design the look and feel for each map layer.

Step 6: Share your map

Use GeoExplorer to save your finished map, and export it to be embeddable in any web page.

Advanced: Make your own mapping application

Use GXP and the included OpenGeo Suite client SDK tools to create and deploy your own custom mapping application.
