OpenGeo Suite Client SDK
========================

The Client SDK provides tools for building web mapping applications backed by 
the OpenGeo Suite.

To see usage, please run the suite-sdk script inside the bin directory.

The most current documentation on how to use this script is available at:
  http://suite.opengeo.org/opengeo-docs/usermanual/tutorials/clientsdk.html

API documentation for the Client SDK is found at:
  http://suite.opengeo.org/opengeo-docs/sdk-api/
